描述
++++++++++++++

给指定邮箱发送邮件。

参数
+++++++++++++++

* 邮箱地址：收件人邮箱地址。
* 邮件主题：设置邮件主题，可以加入变量值。
* 邮件内容：设置邮件内容，可以加入变量值。
